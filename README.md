# cs161b

This repo contains course materials for CS161B. Use the sections below as a map.

## Start here

- `course-overview.md`: narrative walkthrough of the course with links to example files.
- `assignments/`: the assignment specs and any provided assets.
- `practice-exercises.md`: short, hour-scale drills aligned with weekly concepts.
- `INSTRUCTORS_GUIDE.md`: week-by-week teaching plan and in-class activity ideas.

## Examples: two tracks (why there are two folders)

- `examples-from-videos/`: the exact examples shown in the lecture videos, organized by week.
- `examples/`: a parallel, more structured set with weekly READMEs, mini-projects, and a Makefile.

If you are following the videos, use `examples-from-videos/`. If you want a more guided, textbook-style progression, use `examples/`.

## Setup and tooling

- `LINUX_C_CPP_TUTORIAL.md`: terminal basics, compilation, and editor tips on Ubuntu.
- `UBUNTU_VIRTUALBOX_SETUP.md`: VM setup instructions with copy/paste and shared folders.
