# Week 08: Linked Lists

This week adds a simple inventory backed by a linked list.

## New in this week
- `Item` nodes with `next` pointers
- Add/use items from inventory
- Loot tiles on the map

## Build and run

```bash
make
./game
```

## Files
- `map.txt` provides the dungeon layout
- `save.txt` is created when you save
